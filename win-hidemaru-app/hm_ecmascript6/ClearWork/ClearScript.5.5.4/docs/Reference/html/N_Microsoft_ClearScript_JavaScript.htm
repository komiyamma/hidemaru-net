Microsoft.ClearScript.JavaScript Namespace

The Microsoft.ClearScript.JavaScript namespace contains types that provide access to JavaScript-specific features.
Interfaces
  InterfaceDescription

Represents a JavaScript ArrayBuffer.

Defines properties and methods common to all ArrayBuffer views.
IDataView
Represents a JavaScript DataView.
ITypedArray
Defines properties and methods common to all JavaScript typed arrays.
ITypedArrayT
Represents a JavaScript typed array.
